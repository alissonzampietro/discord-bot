# Discord BOT


## Requirements
    - Python >= 3
    - PIP >= 21

## How use the code:
 - Rename *.env-example* to *.env* and input the token of your discord bot in the field *DISCORD_TOKEN*
 - `pip install -r requirements`
 - `python main.py`
